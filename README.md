# GitHub iframe

```html
<iframe src="https://github-iframe.vercel.app/{owner}/{repo}/{path}"></iframe>

<!-- e.g. -->
<iframe
	src="https://github-iframe.vercel.app/nuotsu/sanitypress/src/styles/app.css"
	width="100%"
></iframe>
```
